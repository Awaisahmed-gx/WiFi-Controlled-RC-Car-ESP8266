# WiFi Controlled RC Car using ESP8266

This project is a Wi-Fi controlled RC car built using ESP8266 NodeMCU, L298N motor driver, 18650 batteries, and four DC motors. The car can be controlled wirelessly through a mobile/web interface.

## Components Used
- ESP8266 NodeMCU
- L298N Motor Driver
- 4 DC Motors
- 18650 Battery Pack
- Car Chassis
- Jumper Wires
- Wheels

## Features
- Wireless control using ESP8266
- Forward, backward, left, and right movement
- Speed control through mobile interface
- Four motor driving using L298N
- Battery powered system

## Concepts Applied
- Embedded systems
- Wi-Fi communication
- Motor control
- Circuit wiring
- Power management
- Applied physics concepts

## Project Status
Completed as an academic hardware/IoT project.
